功能说明
---------------------
1.播放页面右上按钮可以下载视频，长按可复制下载链接
2.下载视频无法暂停，退出脚本即可停止所有正在下载的视频
3.数据保存位置:公共目录/SakuraAnime/
4.在播放页面右上可以共享至其他浏览器播放

1.下载管理左划可以删除某个视频或文件夹,右上方可以多选
2.播放本地视频时，右上角共享可以把视频存到相册
3.历史记录界面右上按钮可以清除记录
4.主页面右上按钮可以直接进入下载管理
